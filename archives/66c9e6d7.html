C语言基础 - 循环控制

发表于 2019-01-05 更新于 2019-01-06 分类于 教程

break：退出整个循环
continue：退出本轮循环
在多层嵌套循环中，只能对它所在的那套循环做。

应用1

int n,i;
double sum 0.0;
double sign 1.0;

scanf("%d",&n);
for ( i = 1; i<=n; i++ ）{
    sum += sign/i;
    sign = -sign;
}

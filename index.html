利用git hooks进行自动部署

首先的有一个搭建好的git服务器，在想要自动部署的git库的hook文件中添加自动部署hook，在hook中将DeployDir配置为网站根目录，在网站根目录运行

git clone /usr/gitRepo/web.git

将web文件克隆下来，这样在将更改提交到git仓库的时候，就会自动部署网站。
